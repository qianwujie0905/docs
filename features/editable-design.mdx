# Editable Design

入口位置：页面顶部导航栏中间位置的调色板（palette）图标。点击进入后，当前「整页」都可进行可视化编辑。

在 Superun 的 Editable Design 中，你可以对整个页面进行所见即所得的编辑：选择任意元素直接改动，或用提示微调布局与样式，无需手改代码。


## Credits

- Boxify（将页面结构化为盒模型/可操作分组）会消耗额度。
- 其它编辑（文本、颜色、间距、排版、对齐、圆角、阴影、图片替换等）不消耗额度。

## When to Use Editable Design

适用于：
- 更改文案与多语言占位
- 字体、字号、字重、字距与行高
- 颜色主题、按钮/链接/区块配色
- 布局与栅格（Flex/Grid）、对齐与分布、自动布局
- 间距（内外边距）、圆角、投影、边框
- 响应式断点适配（桌面/平板/移动）
- 图片/图标替换与裁切、插图占位
- 组件级微调与状态样式（Hover/Active/Disabled）

## How to Access and Use Editable Design

1. 点击顶部导航中部的调色板图标进入 Editable Design
2. 将鼠标悬停在目标元素，点击选中；或启用 Boxify 对页面做结构化（仅 Boxify 计费）
3. 使用侧边工具或命令面板进行排版/颜色/布局/间距等编辑
4. 需要大改时可用提示进行批量/联动修改（提示按普通消息计费）
5. 点击 Update prototype 应用修改，即时更新预览与代码
